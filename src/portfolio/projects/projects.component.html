PROJECT


Realtime E-learning SPA using Angular + Spring boot

JUL-AUG 2022
  • A responsive E-learning single page application was created using tools like Angular (HTML, CSS, Typescript, Bootstrap, RXJS) as a front end framework and Spring boot as a backend tool. I have used mySql database to store values.
  • User can register using user registration form and those values are stored in database. This is done by using Restfull web services
  • User can use the registered information to login and can access the courses
  • Admin can add the courses by login into Admin login form
